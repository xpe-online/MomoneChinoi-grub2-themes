# MomoneChinoi-grub2-themes 

> 棒棒喵

A grub theme of 桃寝ちのい. Modified from [vinceliuice/Elegant-grub2-themes](https://github.com/vinceliuice/Elegant-grub2-themes).

## Usage

Clone this repository or download the code zip.

Then:

```sh
./install.sh
```

And reboot your computer, enjoy your new GRUB theme! :)
